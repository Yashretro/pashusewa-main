# PashuSewa (React + Vite)

Animal rescue reporting system built with React and Vite.

## Setup

\\\ash
npm install
echo "VITE_API_BASE_URL=http://localhost:8787" > .env
npm run dev
\\\

Open http://localhost:5173

## Features
- Report form with image upload and geolocation
- Recent reports list with Google Maps links
- Admin dashboard with filtering and distance calculations
